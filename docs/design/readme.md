# Design Notes

This folder is intentionally small. Keep production design guidance in tracked docs and keep exploratory mockups or one-off design drafts local.

## Source Of Truth

- `docs/design-system.md`: themes, typography, spacing, chart colors, motion, and accessibility guidance.
- `docs/design/sizespecifications.md`: menubar icon sizing and pixel-alignment notes.

Older SVG mockups and design exploration files live locally under ignored paths when needed, but should not be committed unless they become durable product documentation.
